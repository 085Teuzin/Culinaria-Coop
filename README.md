# 🔪 Culinaria sob Pressão

​O Culinária Sob Pressão é um Web App de estratégia cooperativa em tempo real, focado em multiplayer local via PeerJS. Ele desafia dois jogadores a coordenarem tarefas em uma cozinha industrial frenética, onde a comunicação é essencial para montar pedidos complexos e gerir o tempo antes que a pressão suba.
